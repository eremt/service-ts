# service-ts

## Getting started

Clone the repository:
```
git clone git@github.com:eremt/service-ts.git
```

Install dependencies:
```
npm install
```

Rename project from `service-ts` to `your-project`:
```
sed -i 's/service-ts/your-project/g' *
```

## Development
Start the development server:
```
npm run dev
```
